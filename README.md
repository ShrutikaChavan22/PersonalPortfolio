# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my skills, projects, certifications, and contact information in a modern and user-friendly design.

## 🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Smooth scrolling navigation
- About Me section
- Projects showcase
- Skills section
- Contact information
- GitHub and LinkedIn profile links
- Resume download button

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
